# Gestao de Planos de Saude

Projeto Flutter desenvolvido para a disciplina de Desenvolvimento para Dispositivos
Moveis I.

O historico do reposititorio sera organizado por branches de iteracao/feature,
seguindo a orientacao do enunciado do trabalho.

## Plano de branches

- `main`: estrutura inicial e versao integrada do projeto.
- `prototipo`: navegacao inicial e telas visuais sem persistencia.
- `iteracao01-modelagem`: dominio, DER e script DDL.
- `iteracao02-infra-banco`: conexao com MySQL/MariaDB e repositories.
- `iteracao03-autenticacao`: login, registro e usuario padrao.
- `iteracao04-cadastros-base`: CRUD de clientes, planos e prestadores.
- `iteracao05-operacoes-plano`: dependentes, contratos, pagamentos e coberturas.
- `iteracao06-relatorios-entrega`: simulacao, relatorios e ajustes finais.
